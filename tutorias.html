Como instalar o GitHub Desktop no Linux

Aprenda passo a passo a instalar o GitHub Desktop no Linux usando comandos no terminal. Ideal para desenvolvedores e usuários que querem gerenciar seus repositórios Git de forma prática.

Comandos de Instalação

  1. Baixar o pacote .deb via terminal: 
    wget https://github.com/shiftkey/desktop/releases/download/release-version/GitHubDesktop-linux-amd64-3.3.12-linux1.deb
  2. Instalar o pacote: 
    sudo apt install ./GitHubDesktop-linux-amd64-3.3.12-linux1.deb
  3. Corrigir dependências (caso apareça erro): 
    sudo apt --fix-broken install
  4. Abrir o GitHub Desktop pelo menu de aplicativos ou terminal: 
    github-desktop

Como configurar AdGuard no iPhone

Aprenda a configurar o AdGuard no iPhone diretamente pelo perfil do site oficial. Ideal para bloquear anúncios e proteger sua navegação.

Passo a Passo

  1. Acesse o site oficial do AdGuard no Safari do seu iPhone.
  2. Baixe o perfil de configuração fornecido pelo site.
  3. Vá em Ajustes → Geral → Perfis e instale o perfil baixado.
  4. Ative o perfil para permitir o bloqueio de anúncios.
  5. Reinicie o Safari ou apps que deseja proteger para aplicar as configurações.
